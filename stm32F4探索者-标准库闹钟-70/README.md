# STM32F4 探索者闹钟系统

## 📌 项目简介
基于 **STM32F4 探索者开发板** 实现的多功能闹钟系统，使用标准外设库 (Standard Peripheral Library) 开发。

## 🛠️ 平台
- **开发板**: 正点原子 STM32F4 探索者 (Explorer)
- **芯片**: STM32F407ZGT6
- **库**: STM32F4 DSP_StdPeriph_Lib

## 📂 包含文件
- `内容/Projects/`: 源码目录。
- `内容/alarmb_code.txt`: 核心逻辑代码片段。
- `题目/设计报告.md`: 详细的设计文档。

## ✨ 功能
- **RTC 时钟**: 利用芯片内部 RTC 实现精准走时。
- **闹钟设置**: 支持设置多组闹铃。
- **按键交互**: 通过开发板按键进行设置与控制。
