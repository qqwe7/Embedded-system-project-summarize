# 温室大棚环境监测系统

## 📌 项目简介
一个基于 51 单片机的智能温室大棚监控系统，用于实时监测环境参数并进行自动控制。

## 🛠️ 系统组成
- **控制器**: 51单片机
- **传感器**: 温湿度传感器 (如 DHT11), 光照传感器等 (推测)。
- **执行器**: 模拟风扇、加热器等。
- **显示**: LCD1602 或 数码管。

## 📂 核心资源
- `内容/ProPrj_温室控制单_*.epro`: 可能是立创EDA或特定工程文件。
- `内容/温室大棚.pdsprj`: Proteus 仿真。
- `内容/Design_Report_v3.pdf`: 详细设计报告。

## ✨ 功能
1. **环境监测**: 实时采集温度、湿度数据。
2. **阈值报警**: 超过设定值触发报警或自动控制设备。
3. **数据显示**: 直观显示当前环境状态。
