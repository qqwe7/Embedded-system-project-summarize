# 8位动态数码管时钟设计

## 📌 项目简介
本项目基于 **51单片机** 设计了一个8位动态数码管时钟系统。系统能够实时显示时、分、秒，并支持时间校准功能。

## 🛠️ 技术栈
- **核心控制器**: 51单片机 (STC89C51/AT89C51)
- **显示器件**: 8位共阴/共阳数码管
- **开发环境**: Keil uVision
- **仿真工具**: Proteus

## 📂 目录结构
- `内容/代码`: Keil工程源码 (`main.c` 等)
- `内容/仿真`: Proteus 电路仿真文件 (`.pdsprj`)
- `内容/report_final_v2.docx`: 项目设计报告

## ✨ 功能特性
1. **时间显示**: 通过数码管动态扫描技术显示 `HH-MM-SS`。
2. **时间调整**: 支持通过按键设置当前时间。
3. **仿真演示**: 提供完整的电路连接与运行仿真。
