# STM32 简易逻辑分析仪

## 📌 项目简介
利用 STM32 的高速 GPIO 读取或 ADC 功能，实现一个简易的逻辑信号分析仪，配合上位机进行波形显示。

## 🛠️ 技术栈
- **下位机**: STM32F103 (Keil 工程)
- **上位机**: Python (`viewer.py`) 用于数据接收与波形绘制。
- **通信**: USB 虚拟串口 / UART。

## 📂 目录说明
- `内容/Project.uvprojx`: STM32 固件工程。
- `内容/viewer.py`: Python 上位机脚本。
- `内容/Hardware/`: 硬件驱动代码。

## ✨ 特性
- 低成本逻辑分析方案。
- 支持多通道数字信号采集。
- 配合 Python 实现数据的可视化分析。
