# STM32 智能导盲杖

## 📌 项目简介
基于 STM32 微控制器的智能导盲杖设计，旨在辅助视障人士安全出行。

## 🛠️ 硬件方案
- **主控**: STM32F103 系列
- **传感器**: 超声波测距模块 (HC-SR04) 用于避障。
- **交互**: 语音播报模块 / 蜂鸣器报警。

## 📂 资源详情
- `导盲杖/`: Keil MDK 工程源码。
- `Project Backups/`: 仿真备份文件。
- `新建文件夹/`: 包含毕业设计论文模板、开题报告及相关素材。

## ✨ 功能
1. **障碍物检测**: 实时检测前方障碍物距离。
2. **多级报警**: 根据距离远近发出不同频率的提示音或语音。
