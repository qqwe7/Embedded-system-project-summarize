# 嵌入式系统开发项目合集 (Embedded Systems Projects)

## 📖 简介 (Introduction)
本项目仓库收录了基于 **8051单片机**、**STM32** (ARM Cortex-M) 以及 **Linux (OrangePi)** 平台的嵌入式开发项目。
这些项目涵盖了从基础的课程设计到复杂的毕业设计，内容包括硬件驱动开发、电路仿真 (Proteus)、传感器数据采集、人机交互界面以及机器视觉应用。

所有项目均按模块化结构组织，通常包含：
- 🛠️ **源码**: Keil C51/MDK 工程或 Python 脚本
- ⚡ **仿真**: Proteus 电路设计与仿真文件
- 📝 **文档**: 设计报告、论文、使用说明书或演示视频

## 📂 项目分类 (Project Categories)

### 🟢 51单片机系列 (MCS-51)
基于传统 8051 架构的基础外设控制与系统设计，适合入门学习。

| 项目名称 | 描述 | 关键技术 |
| :--- | :--- | :--- |
| **8051-8位数码管时钟** | 基于动态扫描的数字时钟设计 | 定时器, 数码管驱动, 动态扫描 |
| **8051-电子秒表** | 精确计时的电子秒表系统 | 外部中断, 定时器, 按键消抖 |
| **8051-数码管显示控制** | 多种模式的数码管显示实验 | GPIO 控制, 驱动逻辑 |
| **8051-温室大棚监控** | 农业环境监测系统 (含报告) | 传感器(温湿度), LCD/数码管显示, 阈值报警 |

### 🔵 STM32 系列 (ARM Cortex-M3/M4)
进阶嵌入式应用，涉及更复杂的逻辑、总线协议及外设驱动。

| 项目名称 | 描述 | 关键技术 |
| :--- | :--- | :--- |
| **STM32-毕设导盲杖** | **(毕业设计)** 智能辅助导盲设备 | 超声波测距, 语音播报, 传感器融合 |
| **STM32-简易逻辑分析仪** | 信号采集与分析工具 | ADC/GPIO 高速采集, Python 上位机通信 |
| **STM32F4-标准库闹钟** | 基于 F4 探索者的时钟/闹钟应用 | RTC (实时时钟), 标准库开发 |
| **STM32F4-屏幕显示** | 屏幕驱动开发大作业 | LCD/OLED 驱动, 图形显示 |

### 🟠 机器视觉与 Linux (Advanced)
基于高性能开发板的视觉应用。

| 项目名称 | 描述 | 关键技术 |
| :--- | :--- | :--- |
| **OrangePi-YOLO视觉识别** | **(毕业设计)** 基于深度学习的目标检测 | Linux, Python, YOLO 算法, OpenCV |

## 🛠️ 技术栈 (Tech Stack)

- **硬件平台**: 
  - STC89C51 / AT89C51
  - STM32F103 / STM32F407
  - OrangePi (香橙派)
- **软件开发**:
  - Keil uVision (C51 / MDK-ARM)
  - VS Code
  - Python (用于上位机或视觉算法)
- **硬件仿真**:
  - Proteus Professional
- **文档工具**:
  - LaTeX / Word (设计报告)

## 📁 目录结构说明 (Directory Structure)

绝大多数项目文件夹遵循以下统一结构：

```text
项目文件夹/
├── 内容/           # 核心交付资源
│   ├── 代码/       # 源代码及工程文件 (Source Code)
│   ├── 仿真/       # 电路仿真文件 (Simulation)
│   └── 文档/       # 报告、论文或视频 (Documentation)
└── 题目/           # 原始需求、任务书或参考资料 (Requirements)
```
