# 51单片机电子秒表

## 📌 项目简介
本项目实现了一个基于 **51单片机** 的电子秒表，精度可达 0.01秒 (或0.1秒)。

## 🛠️ 技术栈
- **MCU**: 51单片机
- **显示**: 数码管
- **IDE**: Keil C51
- **Sim**: Proteus

## 📂 资源说明
- `内容/秒表代码`: 包含 C 语言源码及 Hex 文件。
- `内容/秒表电路图`: Proteus 仿真工程。

## ✨ 主要功能
1. **计时功能**: 启动、暂停、复位。
2. **显示**: 实时显示经过的时间。
3. **中断控制**: 利用定时器中断实现精确计时。
