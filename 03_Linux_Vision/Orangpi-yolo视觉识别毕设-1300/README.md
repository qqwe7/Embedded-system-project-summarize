# OrangePi YOLO 视觉识别系统

## 📌 项目简介
本项目基于 **OrangePi (香橙派)** 开发板，利用 **YOLO (You Only Look Once)** 算法实现实时的目标检测与识别。这是一个高阶的嵌入式机器视觉毕业设计项目。

## 🛠️ 技术栈
- **硬件**: OrangePi 开发板 (Linux ARM)
- **软件**: Python, OpenCV, PyTorch/Darknet (YOLO模型)
- **操作系统**: Armbian / Ubuntu
