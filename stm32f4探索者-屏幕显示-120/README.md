# STM32F4 屏幕显示驱动

## 📌 项目简介
基于 STM32F4 的显示屏驱动大作业，实现了 LCD 或 OLED 的图形/字符显示功能。

## 🛠️ 技术栈
- **MCU**: STM32F407
- **显示屏**: TFT-LCD (FSMC 接口) 或 OLED (SPI/I2C)。
- **框架**: 正点原子 SYSTEM/HARDWARE 架构。

## 📂 目录
- `内容/HARDWARE/`: 屏幕驱动实现 (LCD.c / OLED.c)。
- `内容/USER/`: 主逻辑 `main.c`。
- `题目/`: 包含大作业题目说明。

## ✨ 功能
- 实现了屏幕的底层初始化与画点/画线函数。
- 支持中英文字符显示。
- (可选) 简单的 GUI 交互界面。
